Provides Hydro Raindrop Multi Factor Authentication to your WordPress website.

== Description ==

Allows users to enable the Hydro Raindrop MFA to add an extra layer of security.

For the MFA to work you need to register for an account at https://www.hydrogenplatform.com/developers.

Requirements:

* SSL MUST be enabled for MFA to work.
* PHP 7.1 or higher is required.
